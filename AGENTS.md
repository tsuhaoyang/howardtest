## Cursor Cloud specific instructions

This is a minimal Python 3 demo repository with two standalone scripts (`test1.py`, `test2.py`). There are no dependencies, no package manager, no build system, no tests framework, and no services to run.

- **Run scripts**: `python3 test1.py` and `python3 test2.py` from the repo root.
- **Python version**: Requires Python 3.6+ (f-strings are used in `test2.py`).
- **No lint/test/build tooling** is configured in this repo.
